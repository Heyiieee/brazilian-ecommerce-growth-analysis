# 巴西电商平台经营增长分析

## 📌 项目背景

2018年中平台GMV出现阶段性波动。  
本项目基于Brazilian E-commerce公开数据集，通过结构化拆解方法，对平台经营表现进行系统性分析，定位增长驱动因素及下滑原因，并提出优化建议。

## 🎯 分析目标

- 拆解GMV增长逻辑
- 判断平台增长模式
- 定位2018年6月GMV下滑原因
- 分析新老用户结构
- 输出可执行经营建议

## 📊 分析框架

GMV = 用户数 × 人均订单数 × 客单价

具体拆解路径：

1. 月度GMV趋势分析
2. 订单量与用户规模变化
3. 客单价波动分析
4. 新老用户结构分析
5. 复购率与生命周期判断



## 🔍 核心发现

### 1️⃣ 平台增长模式

平台属于“新客驱动型增长”模式：

- 新客占比长期高于 97%
- 复购率整体低于 3%
- 订单规模高度依赖新增用户

### 2️⃣ 2018年6月下滑原因

- 2018年6月活跃用户下降 9.4%
- 其中新客减少 628 人
- 老客数量基本稳定

结论：

GMV下滑主要由新客获取能力下降导致，而非用户流失或转化率问题。

### 3️⃣ 2018年8月客单价下降

- 用户规模恢复
- 但客单价显著下降

可能原因：

- 商品结构向低价品类倾斜
- 促销策略变化
- 高客单品类销量下滑



## 🚀 经营建议

### 建议一：构建复购体系

- 建立会员体系
- 推动二次营销（优惠券、定向触达）
- 提升老客贡献比例

目标：复购率提升至 10% 以上。

### 建议二：优化拉新策略

- 分析渠道ROI
- 优化高成本低转化渠道
- 提高获客质量而非单纯规模

### 建议三：优化商品结构

- 分析高客单品类趋势
- 控制低毛利促销比例
- 稳定平台客单价水平
